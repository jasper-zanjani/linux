**Target files** are equivalent to SysVInit runlevels.

| SystemD target    | SysVInit runlevel |
| ----------------- | ----------------- |
| poweroff.target   | 0                 |
| rescue.target     | 1                 |
| multi-user.target | 3                 |
| graphical.target  | 5                 |
| reboot.target     | 6                 |
| emergency.target  | emergency         |
