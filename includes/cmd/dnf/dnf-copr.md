Packages in [Copr](https://copr.fedorainfracloud.org/) repositories do not need to follow Fedora Packaging Guidelines.

-   [achno/gowall](https://github.com/Achno/gowall)
-   [atim/lazygit](https://github.com/jesseduffield/lazygit)
-   [petersen/nix](https://copr.fedorainfracloud.org/coprs/petersen/nix/)
-   [solopasha/hyprland](https://copr.fedorainfracloud.org/coprs/solopasha/hyprland/)

```sh
# Install lazygit
dnf copr enable atim/lazygit -y
dnf install lazygit

# Install hyprland
dnf copr enable solopasha/hyprland -y
dnf install hyprland
```
