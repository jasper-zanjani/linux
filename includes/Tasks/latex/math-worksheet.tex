\section{Notes}
 Simplify the following negative exponent problems. You want to get an answer with   all positive exponents.

\begin{multicols}{2}
\begin{enumerate}
\item $\dfrac{1}{k^{-5}}$
\item $k^{-5}$
\item $a^3 b^{-2}$
\item $c^{-4}d^{4}$
\item $(a^2 b^{-7})^{0}$
\item $\dfrac{-45}{5x^{-6}}$
\item $\dfrac{x^{-8}}{4y^{15}}$
\item $\dfrac{2a^3}{b^{-5}}$
\item $\dfrac{y^{-9}}{x^{-15}}$
\item $ - 10y^{-3}$
\end{enumerate}
\end{multicols}
